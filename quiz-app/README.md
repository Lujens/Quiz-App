# Quiz App - Take-Home Kata

A web-based quiz application built with React and TypeScript for the Direct Supply internship application.

## 🎯 Project Overview

This quiz app allows users to:
- Select quiz categories and difficulty levels
- Answer trivia questions from the Open Trivia Database API
- Track their score in real-time
- View performance feedback at the end

## 🛠️ Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety and better developer experience
- **Vite** - Fast build tool and dev server
- **Open Trivia Database API** - Question source (https://opentdb.com/)

## 🚀 Getting Started

### Prerequisites
- Node.js (v20.19.0 or higher recommended)
- npm

### Installation
```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
