# Quiz App – Planning Notes

## Goal
Build a small quiz web app that fetches questions from an external API,
allows users to answer them, and displays a final score.

## Target User
Someone who wants to quickly test their knowledge in a fun, simple way.

## Core Features
- Fetch multiple-choice questions from Open Trivia DB
- Display one question at a time
- Track score
- Show final results

## Nice-to-Have (If Time Allows)
- Difficulty selector
- Score breakdown at the end

## Out of Scope
- Authentication
- Persistent storage
- Multiplayer or real-time features
